<h2>Программная реализация. Описание всех файлов</h2>
<p>Программы для автопилота и математической модели написаны на языке Python с использованием библиотек kRPC, matplotlib, numpy</p>
<ul>
  <li>autopilot.py - программа для автопилота</li>
  <li>ksp_module.py - программа для распаковки всех данных из файла flight_data.txt</li>
  <li>mat_model.py - программа для подсчета математической модели и выведения графиков</li>
  <li>ALTITUDE.png - график зависимости высоты от времени</li>
  <li>SPEED.png - график зависимости скорости от времени</li>
  <li>flight_data.txt - текстовый файл с записанными данными из полета в KSP</li>
  <li>Видео полета.mp3 - видео полета в KSP</li>
  <li>Видео_отчет - предзащита нашего проекта</li>
  <li>Курсовая.docx - курсовая работа по нашему проекту</li>
  <li>Презентация.pptx - презентация по нашему проекту</li>
</ul>

<h2>Цель проекта</h2>
<p>Разработка и практическая реализация в среде KSP программного автопилота на языке Python, способного автоматизировать ключевые этапы космического полёта корабля Восход 1: запуск, выведение на орбиту и посадку</p>
<h2>Задачи проекта</h2>
<ul>
  <li>Настройка симуляционной среды и создание базового интерфейса управления</li>
  <li>Разработка и программирование алгоритма автоматического выведения на орбиту</li>
  <li>Создание системы автоматического управления для спуска и посадки</li>
  <li>Верификация модели, анализ данных и подведение итогов</li>
</ul>
